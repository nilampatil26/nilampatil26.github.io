# Reserver une session

Cette page sert a partager les disponibilites de cours, les horaires de session et les prochains creneaux de tutorat.

Si vous souhaitez reserver une session de tutorat pour votre enfant, contactez-moi via la section de contact et je pourrai vous proposer les horaires les plus adaptes.

## Cette Page Peut Vous Aider A

- Voir les prochaines disponibilites pour les sessions individuelles
- Consulter les mises a jour des horaires de petits groupes
- Planifier un emploi du temps hebdomadaire de tutorat
- Suivre les ateliers speciaux et les sessions de vacances

## Note De Reservation

Pour le moment, les reservations sont gerees manuellement afin que je puisse recommander le format de cours, le groupe d'age et le parcours d'apprentissage les plus adaptes a chaque eleve.
